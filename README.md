# React-Native-Todo
A simple todo application for Android.

# How to use
1. Clone the project
2. $ npm i
3. Connect an android device/emulator
4. $ react-native run-android

# Features
1. Add Todo
2. Edit Todo
3. Delete Todo
4. Mark Todo as done
5. Filter Todo by marked state
6. Search Todo by title
7. Async Storage, Todo is saved on app exit
